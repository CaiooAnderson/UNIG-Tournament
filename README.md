# UNIG-Tournament

> Treinamento pessoal para vaga de Desenvolvedor Front-End Júnior na UNIG

## Sobre o Projeto

Este projeto foi criado para praticar e reforçar conhecimentos essenciais exigidos em vagas de Desenvolvedor Front-End Júnior, utilizando as principais tecnologias do mercado:

- **React** (com Vite)
- **HTML, CSS e JavaScript**
- **Git** para controle de versão
- **npm/yarn** para gerenciamento de pacotes
- **Consumo de APIs RESTful**
- **Design Responsivo e Acessibilidade**

## Objetivos

- Praticar a criação de interfaces modernas e responsivas
- Consumir e integrar APIs RESTful
- Utilizar controle de versão com Git
- Aplicar conceitos de acessibilidade
- Aprender automação e empacotamento com Vite

## Como rodar o projeto

1. Instale o Node.js (https://nodejs.org/)
2. Clone este repositório:
	```bash
	git clone https://github.com/CaiooAnderson/UNIG-Tournament.git
	```
3. Entre na pasta do projeto e instale as dependências:
	```bash
	cd UNIG-Tournament
	npm install
	# ou
	yarn
	```
4. Inicie o servidor de desenvolvimento:
	```bash
	npm run dev
	# ou
	yarn dev
	```
5. Acesse o projeto em `http://localhost:5173`

## Estrutura sugerida

- `src/` — código-fonte React
- `public/` — arquivos estáticos
- `package.json` — dependências e scripts

## Comandos úteis

- `npm run dev` — inicia o servidor de desenvolvimento
- `npm run build` — gera a versão de produção
- `git add . && git commit -m "mensagem"` — versionamento

## Aprendizados e Práticas

- Componentização com React
- Consumo de APIs com fetch/axios
- Estilização com CSS moderno (Flexbox, Grid)
- Responsividade (media queries)
- Acessibilidade (uso de aria, alt, semântica)
